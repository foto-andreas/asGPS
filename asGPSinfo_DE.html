asGPS: Ein Geocoding Plugin für AfterShot Pro

Kostenlose Nutzung und quelloffen

Das Plugin asGPS kann kostenlos benutzt werden. Sie müssen keine Lizenzgebühren dafür bezahlen. Außerdem können Sie den Quelltext kostenlos erhalten und auf Ihre Bedürfnisse anpassen.

Wenn Sie sich bedanken möchten oder eine Tasse Kaffee oder ein Glas Bier spendieren wollen, nutzen Sie den Paypal-Spenden-Button hier. Ein Klick darauf öffnet ein Browser-Fenster, über das Sie sich bei Paypal einloggen oder einen neuen Zugang einrichten können.
